# Сервисы из Practice 2

## Описание

Страница `services.php` теперь содержит 3 сервиса из Practice 2:

### 1. SVG Drawer
**URL:** `services/drawer/drawer.php?num=12345`

**Описание:** Генерирует SVG фигуры из чисел с помощью побитовых операций.

**Как работает:**
- Биты 0-1: тип фигуры (0=прямоугольник, 1=круг, 2=треугольник, 3=эллипс)
- Биты 2-5: индекс цвета (16 цветов)
- Биты 6-11: ширина (20 + значение * 5)
- Биты 12-17: высота (20 + значение * 5)

**Примеры:**
- `drawer.php?num=12345` — серебристый круг
- `drawer.php?num=100` — черный прямоугольник
- `drawer.php?num=999` — цветной треугольник

**Файлы:**
- `services/drawer/drawer.php` — генератор SVG
- `services/drawer/drawer_lib.php` — библиотека функций

### 2. Array Sorter
**URL:** `services/sort/sort.php`

**Описание:** Сортирует массивы чисел алгоритмом сортировки выбором (selection sort).

**Как использовать:**
1. Откройте `services/sort/sort.php`
2. Введите числа через запятую: `5, 2, 8, 1, 9`
3. Нажмите "Sort"
4. Увидите оригинальный и отсортированный массивы

**Файлы:**
- `services/sort/sort.php` — интерфейс сортировки
- `services/sort/sort_lib.php` — библиотека функций

### 3. Admin Commands
**URL:** `services/admin/admin.php`

**Описание:** Выполнение системных команд (только для администраторов).

**Доступные команды:**
- `whoami` — текущий пользователь
- `id` — информация о пользователе
- `ps` — список процессов
- `ls` — список файлов
- `uptime` — время работы системы
- `uname` — информация о системе
- `df` — использование диска
- `free` — использование памяти
- `redis-ping` — проверка Redis

**Требования:** Нужно войти под учетной записью администратора (admin/password123)

**Файлы:**
- `services/admin/admin.php` — интерфейс админ-панели
- `services/admin/admin_lib.php` — библиотека функций
