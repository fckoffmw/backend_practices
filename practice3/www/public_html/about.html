О проекте

Это учебный проект по предмету Backend-разработка — практическая работа №3.

В проекте используется связка nginx + apache + php + MySQL. Статические страницы (вроде этой) обслуживаются nginx, а динамические запросы обрабатывает Apache с PHP.

Цель проекта

Цель — реализовать веб-приложение с базовой авторизацией, REST API и разделением ответственности между веб-серверами.

  • nginx — отдаёт статику
  • apache + php — обрабатывают динамику
  • MySQL — хранит данные пользователей и заказов

Связанные страницы

  • Главная страница
  • Пользователи (API)
  • Заказы (API)
